# 加密文本文件管理器

这是一个基于Python的加密文本文件管理器，具有图形用户界面，支持用户认证、文件加密和文件管理功能。

## 功能特点

- 用户认证系统，需要用户名和密码才能访问
- 文件内容加密存储，保护隐私数据
- 树状文件夹目录浏览
- 文本编辑器支持创建和编辑文本文件
- 支持创建文件夹和文件
- 完全离线运行，数据存储在本地
- 绿色运行，无需安装

## 系统要求

- Python 3.6 或更高版本
- 依赖库：cryptography
- tkinter (Python标准库，无需单独安装)

## 安装与运行

### Python版本运行
1. 确保已安装Python 3.6+
2. 安装依赖库：
   ```
   pip install -r requirements.txt
   ```
3. 运行应用程序：
   ```
   python main.py
   ```

### 打包为可执行文件
1. 安装PyInstaller：
   ```
   pip install pyinstaller
   ```
2. 运行打包脚本：
   ```
   python build.py
   ```
3. 打包完成后，可执行文件位于`dist`目录中
4. 直接运行`dist`目录下的`加密文本管理器.exe`即可使用

## 使用说明

### 首次使用

1. 启动应用程序后，会显示登录界面
2. 点击"注册"按钮，输入用户名和密码创建新账户
3. 注册成功后，使用创建的账户登录

### 文件操作

- 左侧树状视图显示文件夹和文件结构
- 点击文件可在右侧编辑区域打开和编辑
- 通过"文件"菜单可以创建新文件和文件夹
- 编辑完成后，点击"保存"菜单项保存文件
- 点击"退出"可安全退出应用程序

## 安全说明

- 所有文件内容使用Fernet对称加密算法加密存储
- 密码使用SHA-256哈希算法加盐存储，不保存明文密码
- 加密密钥基于用户密码使用PBKDF2派生，不存储在磁盘上
- 每个用户的文件相互隔离，无法访问其他用户的文件

## 数据存储

应用程序在以下位置存储数据：
- 开发环境：
  - 用户信息：`./data/users/`
  - 加密文件：`./data/files/用户名/`
- 打包后的exe：
  - 用户信息：`exe所在目录/data/users/`
  - 加密文件：`exe所在目录/data/files/用户名/`

## 注意事项

- 请妥善保管您的用户名和密码，忘记密码将无法恢复已加密的文件
- 定期备份`data`文件夹以防数据丢失
- 本应用为绿色软件，可以复制整个文件夹到其他位置或设备使用
